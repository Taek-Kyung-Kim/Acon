자바스크립트의 상수(const)

상수 : 변하지 않는 수, 한번 참조하면 다른 것을 참조할 수 없다.

상수(const)를 사용할 때 주의해야 할 점

  • const PI; PI=3.14 (x)(Missing initializer in const declaration 오류)
  • const PI =3.14 (o) : 상수는 선언과 동시에 data를 참조해야한다.
  • 절대 변경되지 않을 거라는 확신이 있는 data를 참조한다.
  • const car_arr=["a","b","c"]; car_array[2]="qm3"; : 만약 참조하는 것이 배열이면 배열을 바꾸는 것은 불가능하나 값을 바꾸는 것은 가능하다. (배열을 선언할떄는 const많이사용)
  • 상수의 이름은 대문자로만 작성한다.(개발자간의 약속) ex) colorBlue (x) => COLOR_BLUE(O)

HTML의 속성

모든 요소는 속성을 가질 수 있다. 속성은 해당 요소가 다른 요소와 차이를 갖는데 도움을 준다.

input 요소의 type과 value 속성

속성의 값은 약속된 것과 약속되지 않은 것이 있다.

속성을 작성하는 규칙

  • 속성은 시작하는 태그에 작성하고 약속된 속성만 사용한다.
  • 작성법: name="value" or name
  • 속성은 복수로 작성할 수 있다.
  • 한 요소에 같은 속성을 두번 사용하면 안된다.
  • value 작성법 : name="value" or name2='value' name3=value
  • name=value로 따옴표를 작성하지 않는 것을 권장하지 않는다.(특히 value에 띄어쓰기가 있으면 절대x
  • name=va lue (x) : 값에 띄어쓰기가 있는경우 절대 따움표를 생략할 수 없다.
  • 참 거짓 속성(Boolean)을 제외한 속성의 모든 value의 type은 문자열이다. 예) name=2021 (javascript "2021" = 문자열이라는 의미 , 2021 => 수 ), (html "2021" = 문자열이라는 의미 , 2021 => 문자열 ) 2021+1=2022 , "2021"+1 = "20211"

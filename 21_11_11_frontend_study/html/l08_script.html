html에서 자바스크립트를 작성하고 DOM(Document Object Model)을 조작하자

작성법 : script 태그를 작성하고 document 객체의 함수를 사용

자바스크립트로 글자를 파란색으로 바꾼다.

브라우저가 DOM을 생성하는 순서와 자바스크립트의 동작

  • 브라우저는 url 주소로 http 통신(요청)을 한 후 html 문서를 받는다(응답)
  • 받은 문서를 해석한다.(DOM tree를 구성한다)
  • html에 작성된 순서대로 document 의 객체(node)를 생성한다
  • 이 떄 script가 생성되지 않은 객체를 참조하면 undefined(정의되지않은) 상태로 오류를 발생시킨다.

자바스크립트로 글자를 파란색으로 바꾼다.

자바스크립트와 프론트엔드

프론트엔드는 브라우저(사용자ux/ui)단을 다룬다.

  • html 문서는 서버에 저장되어 있다.
  • html 문서는 브라우저에 의해 해석된다.
  • style(css) 는 html과 같이 해석되고 html이 생성될 떄 스타일 적용한다.
  • javascript 문서도 html 문서와 같이 서버에 저장된다.
  • javascript 는 html 문서를 해석하고 객체로 만들면서 실행된다.
  • 사용자가 생성된 dom 객체를 다룬다는 것은 javascript를 다룬다는 뜻이다.
    • (브라우저에서 자바스크립트가 동적으로 동작하고 있다.)

Document Object Model(DOM0 Tree (문서 객체 모델)

HTML을 위한 표준 객체 모델로 브라우저의 프로그래밍 인터페이스이다.

  • html 요소(태그)를 Object(객체)로 생성한다.
  • 모든 html 요소의 속성을(attribute)를 Element. 으로 접근할 수 있고, 이를 "property를 구성한다"고 할 수 있다.
  • 모든 html Object를 액세스 하는 방법을 제공한다.(getElementById())
  • 모든 html object의 이벤트 속성을 제공한다.

자바스크립트 문법

자바스크립트의 변수(var(사용x), let, const)

객체(데이터)를 참조하는 것을 변수(참조하는 값이 바뀔 수 있는)

  • 데이터를 재사용하기 위해서(개발단)
  • 거의 대부분의 프로그래밍 언어는 엔진에서 데이터를 관리하는데 사용되지 않는 데이터를 삭제하는 시스템(garbage collection)이 존재한다. 이때 변수가 참조하는 데이터는 다시 사용될 수 있다고 생각한다.(시스템단)
  • 가독성을 높인다.(data를 참조하는 변수의 이름)
  • data가 재사용되기 때문에 중복될 걱정이 없다.
  • data가 중복되지 않기 때문에 유지보수가 용이하다.

자바스크립의 함수(function)

자바스크립트의 함수는 일반 프로그래밍에서 사용되는 함수와 동일하다.

  • function 이름(매개변수, 매개변수2...){실행;, 실행2; ... return 반환값 }
  • 매개변수와 반환값은 생략할 수 있다.
  • 이름은 중복될 수 없다.
  • document, window, id, formName 같은 약속된 이름을 사용할 수 없다.
  • 매개변수(parameter)는 let var을 생략할 수 있고 함수 내부({})에서만 사용된다.
  • 함수를 실행할때는 "함수이름();" 을 작정하면 되고 함수에 반환값이 지정되어 있으면 값을 반환한다.
  • 매개변수는 함수가 실행될 떄 값을 지정한다.sum();
  • 여러 실행을 묶어 놓은 집합으로 재사용을 목적으로 한다.

html javascript의 약속된 객체

  • window : 브라우저 객체(BOM)
  • document : html 문서 객체(DOM)
  • String : 개발자에게 필요한 문자열을 제공
  • Number : 개발자에게 필요한 수를 제공
  • boolean : 개발자에게 필요한 참과 거짓 거짓 제공
  • Array : 개발자에게 필요한 배열을 제공
  • Date : 개발자에게 필요한 문자열을 제공li>
  • Math :개발자에게 필요한 연산을 제공

document가 제공하는 요소를 찾기위한 함수(return Element or HTML collection)

HTML Collection : 복수의 Elements가 참조될 떄 도움을 주는 객체

  • document.getElementsByTagName("tagName")=>return HTMLCollection
  • document.getElementsByClassName("className")=>return HTMLCollection
  • document.querySelectorAll("cssSelector")=>return HTMLCollection
  • document.querySelector("cssSelector")=>return Element (선택된 복수의 elements 중 처음 한개만 참조한다.
  • document.getElementById("id") => return Element

html style(속성)로 요소를 꾸미자

illine css 규칙 : style="name:value; name2:value; name3=value"

css는 요소의 레이아웃, 색상, 텍스트, 크기, 이미지, 배경,...등 화면에 표시되는 모든 방식을 제공한다.

html 내부 css 적용방법

  1. head 태그 안에 style 태그를 작성
  2. style 태그안에 seletor {}를 작성
  3. {}안에 name:value; property:value; 을 작성
  4. 내부 css는 인라인 css보다 우선 순위가 낮다

bolck 요소의 style

block(div)

content의 높이(hight)와 넓이(width)

요소의 내용의 넓이와 높이는 widht와 hight로 지정할 수 있다.

padding은 content 주위의 공간

  • padding: top right bottom left; 으로 컨텐츠 주위의 공간을 따로 지정할 수 있다.
  • padding: value; top right bottom left를 한꺼번에 지정할 수 있다.
  • padding-(top, right, bottom, left): value;로 한개의 속성만 지정할 수 있다.
  • padding: top(right left) bottom;
  • padding: (top bottom) (right left);

border(테두리)는 요소의 padding과 margin 사이의 경계선을 작성할 수 있다.

  • border-width: 테두리의 너비를 지정가능
  • border-color: 테두리의 색을 지정가능
  • border-style: 테두리의 스타일을 지정가능
    • solid-실선
    • dotted-점선
    • dashed-절취선
    • duble-이중 실선
    • groove,ridge,inset,outset-3d 효과
    • none-정의하지 않음
    • hidden-숨겨짐
  • border: width style color; 속기
  • border(top right bottom left): width style color; 속기
  • border-radius: 단위(%제외); 사각형인 박스모델의 끝을 둥글게 만든다.
  • border-radius: %; 사각형인 박스모델을 둥글게 만든다.

margin 테두리 밖의 여백(padding과 작성법이 같다)

margin은 padding과 작성법이 같지만 auto라는 value로 box를 정렬할 수 있다.(default:margin-right:auto;)

  • margin-right:auto; block 요소는 컨테이너(중첩된상위 요소)의 가로를 모두 차지하려한다. 이때 가로길이가 컨테이너보다 작으면 나머지를 여백으로 채운다. 이때 auto가 여백에 해당한다.
  • margin-right: auto; 블럭의 왼쪽 정렬
  • margin-left: auto; 블럭의 오른쪽 정렬
  • margin: top auto bottom; 블럭의 중앙 정렬
  • margin: (top bottom) auto; 블럭의 중앙 정렬
  • margin-left: auto; margin-right:auto; 중앙의 중앙 정렬

select 요소는 드롭 다운 목록을 작성하는 데 사용된다.

select 요소는 사용자 입력을 수집하기 위해 양식에서 가장 자주 사용된다.

  • name 속성은 양식이 제출된 후 양식 데이터를 참조하는 데 필요.(name속성을 생략하면 드롭다운 목록의 데이터가 제출되지 않음)
  • id 속성은 드롭다운 목록을 레이블과 연결하는 데 필요합니다.
  • option은 select요소 내부의 사용가능한 옵션을 정의합니다.

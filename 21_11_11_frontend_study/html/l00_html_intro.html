HTTP(Hyper Text Transfer Protoclo)와 HTML(Hyper Text Markup Language

  1. html은 htpp 통신을 할 때 요청(cleant or browser)하는 페이지 서버 응답하는 문서이다.
  2. http 통신은 server와 client가 통신을 하는 규약으로 요청과 응답의 형식을 제공한다.

HTML 기본 작성법

  1. html은 마크업 언어로 태그를 만들고 내용을 작성한다.
  2. html 태그는 약속된 이름이 존재하고 열린태그는 반드시 닫아야한다.<>, < / >
  3. html 문서는 트리구조로 최소위 태그로 html이 존재해야한다.
  4. 태그는 소문자로 작성하는 것을 권장하고, 약속된 이름만 사용한다.
  5. html5는 가장 최신 html 문서 규칙으로 문서 내부의 오류를 자동으로 수정한다.
html 외부에 작성된 내용은 오류로 간주되어 자동으로 내부로 편입된다.

자바스크립트의 배열과 반복문

[]복수의 데이터를 참조하는 배열

  • 선언법 : let car_arr=[value, value, value];....
  • 선언법 : let car_arr=[[],car_arr[0]=value, car_arr[1]=value;....
  • 선언법 : let car_arr=new Array(value,value,value,....);
  • lenght : 배열의 길이 (1부터 시작)
  • 배열은 data를 순서를 이용해서 그룹핑하는 것으로 이때 순서가 key 가 된다. (0부터 시작)
  • 배열을 참조하는 변수에 [key]를 쓰면 해당하는 위치에 저장된 data가 반환된다. 예)car_arr[2]=return "gv80"
